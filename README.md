## Run

### Web

Use any compatible web server, for example

    python -m http.server

Open `localhost:8000` in a browser.

### Electron

Run `npm run start`


### Android

Refer to [qualli-capacitor](https://github.com/niklassc-xyz/qualli-capacitor).
